# Anvil Enchantment Mod

一个 Minecraft Fabric 模组，允许铁砧跨物品类型附魔。

## 功能

- **跨类型附魔**：可以将任何物品的附魔转移到任何其他物品上
- 例如：
  - 将剑上的锋利附魔转移到胸甲上
  - 将镐子上的效率附魔转移到剑上
  - 将附魔书的附魔应用到任何物品上

## 使用方法

1. 将模组放入 `mods` 文件夹
2. 启动游戏
3. 在铁砧中：
   - 左侧放入目标物品（你想附魔的物品）
   - 右侧放入来源物品（带有附魔的物品或附魔书）
   - 不再出现红叉！附魔可以直接转移

## 安装要求

- Minecraft 1.20.1
- Fabric Loader 0.16.9 或更高版本
- Fabric API

## 构建方法

```bash
./gradlew build
```

构建后的模组文件将位于 `build/libs/` 目录下。

## 注意事项

- 附魔等级会正常合并（相同等级+1，不同等级取最高）
- 互斥附魔仍然无法共存（如锋利和亡灵杀手）
- 经验消耗按正常铁砧规则计算
